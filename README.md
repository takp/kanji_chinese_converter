# KanjiChineseConverter

Convert Japanese Kanji to Chinese character.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'kanji_chinese_converter'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install kanji_chinese_converter
